## 10. 自动部署要求

- GitHub 仓库：GITHUB_OWNER/GITHUB_REPO
- 发布分支：main
- Pages 发布目录：/
- 自定义域名：example.com
- DNS 服务商：Dynadot
- DNS 记录：根域名配置 GitHub Pages A 记录，按选择配置 AAAA 记录，www 配置 CNAME 到 GitHub Pages 用户域。
- 脚本安全：GitHub 登录使用 GitHub CLI，Dynadot API Key 从环境变量 DYNADOT_API_KEY 读取，不写入仓库。
